# Prime Cinemas Redesign

A modern UI redesign of the Prime Cinemas Jordan website.

## Features

- Responsive layout
- Modern glassmorphism design
- Movie cards
- Hero carousel
- Smooth animations
- Improved navigation

## Built With

- HTML5
- CSS3
- JavaScript
